Support

    

We will continually make improvements and fix bugs where they can be identified. The results of these efforts will be incorporated into upcoming versions of the software.

In any case, we would like to offer you our full support. Please do not hesitate to contact PicoQuant if you need assistance with attempts to write your own software using PicoQuant's APIs.

If you observe any errors or bugs, please try to find a reproducible error situation. Please e-mail us a detailed description of the problem and of any relevant circumstances, together with the data in question. When you contact the support, please bring up the support request dialog (for example Help | Getting Support… from the main menu) and press the "Export" button of this dialog to export its contents to a text file. Please attach this file when you mail your support request to: support@picoquant.com This will help us with the diagnosis of the problem. Thank you very much in advance. Your feedback will help us to improve the product and its documentation.

Of course, we also appreciate good news: If you have obtained exciting results or published a scientific paper, we would like to know! Please send us an e-mail containing the appropriate citation to info@picoquant.com. Gain additional publicity! PicoQuant maintains a database of publications mentioning PicoQuant devices and software. It can be found at our website at   http://www.picoquant.com/scientific. It is a valuable source if you would like to know which laboratories are using PicoQuant products or how broad the field of various applications is.
